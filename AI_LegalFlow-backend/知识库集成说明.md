LegalFlow 已成功集成专业的法律知识库系统！知识库包含以下内容：

### 4. 相关法律文件（PDF）
- 中华人民共和国民法典
- 合同行政监督管理办法

## API 接口

### 知识库统计
```
GET /api/knowledge/stats
```
返回知识库统计信息和文档列表

### 文档管理
```
GET /api/knowledge/documents          # 获取所有文档
GET /api/knowledge/documents/{id}    # 获取单个文档
```

### 检索接口
```
GET /api/knowledge/search?query=违约金&topK=10&category=法律问答
POST /api/knowledge/search
```
支持关键词检索、分类过滤、标签过滤

### 分类列表
```
GET /api/knowledge/categories
```

## 使用示例

### JavaScript 示例

```javascript
// 检索知识库
async function searchKnowledge(query) {
    const response = await fetch(`/api/knowledge/search?query=${encodeURIComponent(query)}&topK=5`);
    const results = await response.json();
    return results;
}

// 获取知识库统计
async function getStats() {
    const response = await fetch('/api/knowledge/stats');
    const stats = await response.json();
    console.log('知识库文档总数:', stats.totalDocuments);
    return stats;
}

// 使用示例
searchKnowledge('违约金')
    .then(results => {
        results.forEach(r => {
            console.log(`[${(r.score * 100).toFixed(1)}%] ${r.title}`);
            console.log(r.content.substring(0, 100) + '...');
        });
    });
```

### Java 示例

```java
@Autowired
private VectorSearchService vectorSearchService;

// 在Agent中使用
String context = vectorSearchService.buildContextForAgent("合同违约金", 5);
```
