# LegalFlow 项目配置说明

## JDK 17 配置指南

本项目需要 JDK 17，已安装至 `E:\Workspace-java\jdk17`。

### IntelliJ IDEA 配置

1. **打开项目设置**
   - File → Project Structure (Ctrl+Alt+Shift+S)

2. **配置 Project SDK**
   - 选择 "Project" 标签
   - 点击 "SDK" 下拉框
   - 选择 "Add SDK" → "Download JDK..."
   - 选择版本 17 或指向 `E:\Workspace-java\jdk17`

3. **配置 Module SDK**
   - 选择 "Modules" 标签
   - 确保 SDK 设置为 17

4. **配置 Maven/Gradle**
   - File → Settings → Build, Execution, Deployment → Build Tools → Maven/Gradle
   - 设置 Maven/Gradle 的 JDK 为 17

### 命令行运行

```powershell
$env:JAVA_HOME = "E:\Workspace-java\jdk17"
$env:PATH = "$env:JAVA_HOME\bin;$env:PATH"
cd "E:\Workspace-java\AI_LegalFlow\AI_LegalFlow-backend"
mvn spring-boot:run
```

## 环境依赖

项目需要以下服务运行：

- **PostgreSQL 14+**: 数据库（可选，当前为内存模式）
- **Redis**: 缓存和消息队列（可选）
- **RabbitMQ**: 消息队列（可选）
- **Milvus/Qdrant**: 向量数据库（可选）

## 大模型 API 配置

### 阿里云百炼（通义千问）- 推荐

在 `application.yml` 中配置：

```yaml
spring:
  ai:
    model-type: openai
    openai:
      api-key: ${ALIYUN_DASHSCOPE_API_KEY:your-api-key}
      base-url: https://dashscope.aliyuncs.com/compatible-mode/v1
      chat:
        options:
          model: qwen-plus
          temperature: 0.7
```

**获取 API Key：**
1. 访问阿里云百炼控制台：https://bailian.console.aliyun.com/
2. 创建 API Key (DashScope API Key)
3. 在环境变量中设置：`ALIYUN_DASHSCOPE_API_KEY=your-key`

**支持的模型：**
- `qwen-turbo` - 快速响应
- `qwen-plus` - 平衡性能（默认）
- `qwen-max` - 最强模型

### 常见问题

**1. 依赖下载失败**
```bash
mvn dependency:resolve
```

**2. JDK 版本不匹配**
确保项目使用 JDK 17：
```bash
java -version
```

**3. 编译错误**
```bash
mvn clean compile
```

**4. Lombok 不工作**
在 IDEA 中：
- Install Lombok plugin
- Enable annotation processing: Settings → Build → Compiler → Annotation Processors → Enable
